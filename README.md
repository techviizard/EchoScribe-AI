# 🎙️ EchoScribe AI

EchoScribe AI is an intelligent speech-to-text web application that transforms audio into accurate, readable transcripts using AI. Users can upload audio files or record directly from their browser, generate AI-powered summaries, and export transcripts in multiple formats.

🌐 Live Demo: https://echoscribeai-1084744407771.asia-southeast1.run.app/

---

## ✨ Features

### 🎤 Audio Input
- Record audio directly from your browser
- Upload existing audio files
- Supports multiple audio formats:
  - MP3
  - WAV
  - M4A
  - AAC
  - OGG
  - FLAC
  - and more

### 📝 AI Transcription
- Fast and accurate speech-to-text
- Speaker/Dialog-wise transcription
- Paragraph/MOM (Minutes of Meeting) style transcription
- Handles long audio recordings

### 🤖 AI Summarization
Generate intelligent summaries such as:
- Executive Summary
- Minutes of Meeting (MoM)
- Key Highlights
- Action Items
- Bullet Point Summary
- Custom AI Summary

### 📄 Export Options
Download transcripts as:
- TXT
- PDF

### 🔴 Live Meeting Transcription
- Real-time speech recognition
- Background transcription during meetings
- Useful for:
  - Online Meetings
  - Lectures
  - Interviews
  - Seminars
  - Podcasts
